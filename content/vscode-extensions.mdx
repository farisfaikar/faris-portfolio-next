## Extensions untuk Produktivitas

### 1. Git Graph

Git Graph adalah extension yang memungkinkan visualisasi riwayat repository Git dalam bentuk grafis. Dengan Git Graph, pengguna dapat melihat cabang, commit, dan merge secara lebih mudah dan intuitif, sehingga mempermudah navigasi dan pengelolaan proyek Git.

![Untitled](/blogs/code/1.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph](https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph)

### 2. Error Lens

Error Lens adalah extension yang me[no](https://youtu.be/qWwb8S02f_c?si=ioLd3HSkzNtKV_3B)long developer dengan menampilkan pesan kesalahan dan peringatan secara langsung di codebase. Pesan ini ditampilkan secara visual di samping baris kode yang bermasalah, sehingga memudahkan identifikasi dan perbaikan bug.

![Untitled](/blogs/code/2.png)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)

### 3. GitLens

GitLens memperluas kemampuan Git di dalam Visual Studio Code dengan menyediakan informasi rinci tentang commit, penulis, dan perubahan kode. Extension ini juga memudahkan pelacakan perubahan historis dan memberikan wawasan mendalam tentang kode yang sedang dikerjakan.

![Untitled](/blogs/code/3.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)

### 4. Tailwind Fold

Tailwind Fold adalah extensive yang membantu pengguna dalam mengelola class utility dari Tailwind CSS dengan memungkinkan pengguna untuk menyembunyikan atau "fold" class yang panjang. Ini membantu menjaga kode tetap bersih dan mudah dibaca.

![Untitled](/blogs/code/4.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=stivo.tailwind-fold](https://marketplace.visualstudio.com/items?itemName=stivo.tailwind-fold)

### 5. Indent Rainbow

Indent Rainbow adalah extension yang mewarnai indentasi dalam kode untuk memudahkan pembacaan struktur kode. Setiap level indentasi diberi warna yang berbeda, sehingga pengguna dapat dengan cepat memahami hirarki dan struktur blok kode. Anda bisa mengkombinasikan extension ini dengan indent-rainbow-palette untuk mengkustomisasi warna palet indentasi.

![Untitled](/blogs/code/5.png)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow)

![Untitled](/blogs/code/6.png)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=evondev.indent-rainbow-palettes](https://marketplace.visualstudio.com/items?itemName=evondev.indent-rainbow-palettes)

### 6. TabOut

TabOut adalah extension yang memungkinkan pengguna untuk melompat keluar dari tanda kurung, tanda kutip, dan elemen serupa dengan menekan tombol Tab. Ini mempercepat penulisan kode dengan mengurangi jumlah penekanan tombol yang diperlukan untuk navigasi di dalam kode.

![Untitled](/blogs/code/7.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=albert.TabOut](https://marketplace.visualstudio.com/items?itemName=albert.TabOut)

### 7. Auto Rename Tag

Auto Rename Tag adalah extension yang secara otomatis mengganti nama tag HTML atau XML yang dipasangkan ketika salah satu tag diubah. Dengan extension ini, saat pengguna mengedit tag pembuka atau penutup, tag yang berpasangan akan secara otomatis disesuaikan, sehingga menghemat waktu dan mengurangi risiko kesalahan dalam penulisan kode struktur dokumen yang kompleks.

![Untitled](/blogs/code/8.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)

### 8. Subword Navigation

Subword Navigation memungkinkan pengguna untuk menavigasi dan mengedit bagian kata di dalam variabel atau identifier dengan lebih mudah. Ini mempermudah pengeditan kata majemuk atau nama variabel panjang tanpa harus menghapus atau menambah karakter satu per satu.

![Untitled](/blogs/code/9.gif)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=ow.vscode-subword-navigation](https://marketplace.visualstudio.com/items?itemName=ow.vscode-subword-navigation)

### 9. TODO Tree

TODO Tree adalah extension yang membantu pengguna menandai dan melacak catatan TODO, FIXME, dan catatan penting lainnya di dalam kode. Extension ini akan menyoroti baris yang berisi catatan tersebut, sehingga pengguna dapat dengan mudah menemukannya dan menindaklanjutinya.

![Untitled](/blogs/code/10.png)

Link Extension: [[https://marketplace.visualstudio.com/items?itemName=wayou.vscode-todo-highlight](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree)](<[https://marketplace.visualstudio.com/items?itemName=wayou.vscode-todo-highlight](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree)>)

### 10. Toggle Quotes

Toggle Quotes adalah extension yang memungkinkan pengguna untuk dengan mudah mengganti jenis tanda kutip di sekitar string, dari tanda kutip tunggal ke tanda kutip ganda atau sebaliknya. Ini berguna untuk menyesuaikan format kode sesuai dengan konvensi atau preferensi proyek.

![Untitled](/blogs/code/11.png)

Link Extension: [https://marketplace.visualstudio.com/items?itemName=BriteSnow.vscode-toggle-quotes](https://marketplace.visualstudio.com/items?itemName=BriteSnow.vscode-toggle-quotes)
